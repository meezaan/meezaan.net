Title: Developer/Programmer

Reports to: Manager of Technology Services

Based at: Dubai / Sharjah, United Arab Emirates.

Job purpose:

To help existing programmers with coding; to look at existing code and make minor changes and tweaks in sofware or web sites; to code the creation of new software, to customize existing software.

Requirements:

  • Must be an expert in PHP & ASP or ASP.net and VB.net
  • Must have thorough knowledge of DBMS, and/or be able to connect code in the above mentioned languages to MySQL, Access, and/or MS SQL database
  • Must be able to write XHTML and CSS Compliant code
  • Must speak English
  • Must be able to manage time well
  • Must be a hard worker and a team player

Key responsibilities and accountabilities:

  • Code on current projects
  • Edit existing web sites and act as web master for a portfolio of web sites
  • Tweak existing software and customize it as per client needs and requirements
  • Manage the programming of database driven applications
  • Write and develop object oriented code on new and upcoming projects
